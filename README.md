# Boncatta

Boncatta is an independent GitHub Pages site for the online version of 暴塔.

Site URL:

https://boncatta.github.io/

Current features:

- Online room lobby
- WebRTC 1v1 battle sync
- Action target selector, prepared for future multiplayer expansion
- Public random roll, hit range, skill, and target display for every action
- Character compendium with skill probabilities and effect notes
